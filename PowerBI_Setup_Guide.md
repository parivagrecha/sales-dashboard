## Step 2 — Build the Data Model

In **Model view**:

| From Table | From Column | To Table | To Column | Cardinality |
|---|---|---|---|---|
| Raw Data | customer_id | RFM Segmentation | customer_id | Many-to-One |
| Raw Data | category | Category Performance | category | Many-to-One |
| Raw Data | region | Regional Analysis | region | Many-to-One |

**Create a Date Table** (in DAX editor):
```dax
Date = CALENDARAUTO()
```
Then add columns:
```dax
Year = YEAR('Date'[Date])
Month = MONTH('Date'[Date])
MonthName = FORMAT('Date'[Date], "MMM")
Quarter = "Q" & QUARTER('Date'[Date])
```
Relate: `Raw Data[order_date]` → `Date[Date]`

## Step 3 — DAX Measures

Create a new table called `_Measures` and add all measures there.

### Core KPIs
```dax
Total Revenue = SUM('Raw Data'[revenue])

Total Profit = SUM('Raw Data'[profit])

Profit Margin % = 
DIVIDE([Total Profit], [Total Revenue], 0) * 100

Order Count = COUNTROWS('Raw Data')

Avg Order Value = 
AVERAGEX('Raw Data', 'Raw Data'[revenue])

Unique Customers = 
DISTINCTCOUNT('Raw Data'[customer_id])
```

### Time Intelligence
```dax
Revenue Growth MoM = 
VAR CurrentRevenue = [Total Revenue]
VAR PrevMonthRevenue = 
    CALCULATE(
        [Total Revenue],
        DATEADD('Date'[Date], -1, MONTH)
    )
RETURN
    DIVIDE(CurrentRevenue - PrevMonthRevenue, PrevMonthRevenue, 0)

YTD Revenue = 
TOTALYTD([Total Revenue], 'Date'[Date])

Revenue vs Last Year = 
CALCULATE(
    [Total Revenue],
    SAMEPERIODLASTYEAR('Date'[Date])
)

Revenue Growth YoY = 
DIVIDE(
    [Total Revenue] - [Revenue vs Last Year],
    [Revenue vs Last Year],
    0
)
```

### Rankings
```dax
Product Revenue Rank = 
RANKX(
    ALL('Raw Data'[product]),
    [Total Revenue],
    ,
    DESC,
    DENSE
)

Top N Products Flag = 
IF([Product Revenue Rank] <= 10, "Top 10", "Others")
```

Add these slicers on every page (sync them via **View → Sync Slicers**):

|---|---|---|

| Quarter | Date[Quarter] | Tile |

Enable **cross-page filtering**: View → Sync Slicers → select all pages

## Step 7 — Publish

1. **File → Publish → Power BI Service**
2. Select your workspace
3. In the browser, open your report
4. **Pin** key visuals to a Dashboard
5. Set up **Scheduled Refresh** if using live data

## Kaggle Dataset Integration

To use the real Superstore dataset instead of synthetic data:

1. Download from: https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
2. In Power BI: **Transform Data** → rename columns:
   - `Sales` → `revenue`
   - `Profit` → `profit`  
   - `Order Date` → `order_date`
   - `Category` → `category`
   - `Region` → `region`
   - `Product Name` → `product`
3. All measures and visuals will work automatically.
